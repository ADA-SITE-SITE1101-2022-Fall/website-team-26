About the Hour Of Code event

This year, around 200 School of Information Technology and Engineering students from ADA University have conducted one hour coding lesson in around 50 different schools of Azerbaijan. It was done as a part of the global Hour of Code movement.

We, as the Team 26 delivered a lesson in the Baku Secondary School No. 39 to a group of 9th grade students.

This is a team website built on GitHub using HTML, CSS, and JavaScript.
